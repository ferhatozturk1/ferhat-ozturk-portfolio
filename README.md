# Ferhat Öztürk Portfolio

Modern personal portfolio website for Ferhat Öztürk, built with Next.js, TypeScript, Tailwind CSS, Framer Motion and lucide-react.

## Features

- Responsive one-page portfolio layout
- Dark mode by default with light mode support
- Sticky navigation with active section state
- Animated hero section with abstract AI/developer visual
- Featured projects, technical skills, certificates and education sections
- Filterable certificates
- Frontend-only contact form with toast notification
- SEO and Open Graph metadata

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Production

```bash
npm run build
npm run start
```
